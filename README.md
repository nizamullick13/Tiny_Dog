# TinDog 🐶

A responsive landing page for **TinDog** — a playful, fictional dating app for dogs. Built with **Bootstrap 5** and a touch of custom CSS.

## Features

- Fully responsive layout (Bootstrap 5.3 grid)
- Animated gradient hero and footer background
- Sections: hero with app-store buttons, feature highlights, testimonial, three-tier pricing, and footer
- No build step — pure HTML + CSS + Bootstrap via CDN

## Run it

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```

## Structure

```
index.html      # all page sections
css/style.css   # custom styles (gradient animation, icon squares, avatar)
images/         # app mockups, press logos, testimonial photo
```
